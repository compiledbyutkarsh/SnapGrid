# 📸 SnapGrid — Instagram UI Clone

A pixel-perfect Instagram UI clone built with React and Tailwind CSS. Fully responsive with dark theme.

## ✨ Features
- 🏠 Home feed with stories, posts, like & save
- 🔍 Explore page with photo grid
- 💬 Messages with live chat UI
- 🔔 Notifications
- 👤 Profile page with highlights & post grid
- ❤️ Double-tap to like animation
- 📱 Responsive sidebar

## 🛠️ Tech Stack
| Layer | Technology |
|---|---|
| Framework | React.js |
| Styling | Tailwind CSS |
| Icons | Lucide React |
| Build | Vite |
| Deploy | GitHub Pages |

## 🚀 Run Locally

```bash
git clone https://github.com/compiledbyutkarsh/SnapGrid.git
cd SnapGrid
npm install
npm run dev
```
